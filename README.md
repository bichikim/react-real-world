# react-real-world

## Project setup
```
yarn install
# or
lerna bootstrap

```

### Generate favicons
```
yarn favicongen
```

### Compiles and hot-reloads for development


```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Run unit tests
```
yarn test
```

### Lints and fixes files
```
yarn format
```
